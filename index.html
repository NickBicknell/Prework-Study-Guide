HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Elements -
    • An element is a type of content on webpage
    • Elements commonly used to add content to pages:
      • Text elements
      • Image elements
      • Header Elements
      • List elements

      • Meta-related elements

  • Metadata
    • Data about our webpage that the browser uses to effectively serve our webpage to users
    • Typically not seen by users, but it contains important information about accessibility, search engines, and performance
  • Search Engine Optimization (SEO)
    • The process of improving the ranking of a website in search engines such as google or bing
    • Better SEO results in your website more frequently showing up in searches
  • Parent element
    • The element that holds all child elements
  • Child element
    • Elements that are nested inside other elements (parent elements)

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Do Not Repeat Yourself (DRY)
    • A coding principle that encourages devs to reuse or share pieces of code in order to reduce the number of lines of code that needs to be written or maintained
    • DRY can greatly reduce the overall size and complexity of a codebase
  • External CSS style sheet
    • Allows the dev to keep all their CSS rules in a separate file, which makes design changes easier
  • Internal style sheet
    • Contains CSS rules for the webpage in the head section of the HTML file
    • The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in thew page code
  • Inline CSS
    • Style an element directly in the HTML file
    • Useful for quick and permanent changes but are less flexible than external and internal style sheets
